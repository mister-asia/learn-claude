# learn-claude

A simple digital clock web app built as a learning project.

## Features

- Displays the current time in `HH:MM:SS` format
- Updates every second in real time

## Getting Started

Open `index.html` in a browser — no build step required.

## Testing

```bash
npm install
npm test
```

Tests use [Jest](https://jestjs.io/) with a jsdom environment.
