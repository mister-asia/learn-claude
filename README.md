# learn-claude

A simple digital clock web app built as a learning project.

## Stack

- [Next.js](https://nextjs.org/) (App Router)
- TypeScript
- CSS Modules

## Features

- Displays the current time in `HH:MM:SS` format
- Updates every second in real time

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in a browser.

## Testing

```bash
npm test
```

Tests use [Jest](https://jestjs.io/) and cover the `formatTime` utility.
